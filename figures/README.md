# Figures Directory

This directory contains visualizations generated from the analysis notebooks.

Figures are automatically generated when running notebooks.

## Generated Figures

- target_distribution.png - Distribution of Scope 1 & 2 emissions
- correlation_heatmap.png - Feature correlation matrix
- feature_importance.png - Top 10 feature importances
- prediction_vs_actual.png - Model predictions vs actual values
- residual_analysis.png - Residual plots
- sector_emissions.png - Emissions by sector
- geographic_patterns.png - Emissions by country/region

All figures are saved in high resolution (300 DPI) for presentation purposes.
